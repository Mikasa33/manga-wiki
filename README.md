# Manga Wiki

一个使用 [Nuxt](https://nuxt.com/) 开发的漫画 Wiki 网站，同时支持 PC 和移动端。

## Scripts

```bash
# 安装
pnpm install

# 启动
pnpm dev

# 构建
pnpm build

# 预览
pnpm preview
```
